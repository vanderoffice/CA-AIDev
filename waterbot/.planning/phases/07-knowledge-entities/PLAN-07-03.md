# PLAN-07-03: Regional Water Quality Control Boards - Overview

## Objective
Create foundational knowledge about the Regional Water Board system, including their structure, authority, and relationship to the State Water Board.

## Deliverables

### File 1: regional-boards/rwqcb-overview.md
**Purpose:** Overview of the Regional Water Board system
**Content:**
- Nine-region structure established by Porter-Cologne
- Regional Board authority and responsibilities
- Relationship to State Water Board
- Board membership (9 members per region, appointment process)
- Executive Officer role
- Basin Planning authority
- Permitting and enforcement jurisdiction

### File 2: regional-boards/regional-boundaries.md
**Purpose:** Geographic boundaries and hydrologic basis
**Content:**
- How boundaries were determined (watershed-based)
- Region 1: North Coast
- Region 2: San Francisco Bay
- Region 3: Central Coast
- Region 4: Los Angeles
- Region 5: Central Valley (largest)
- Region 6: Lahontan
- Region 7: Colorado River Basin
- Region 8: Santa Ana
- Region 9: San Diego
- Map reference and boundary descriptions

### File 3: regional-boards/regional-state-relationship.md
**Purpose:** How Regional and State Boards interact
**Content:**
- State Board oversight authority
- Appeal process to State Board
- Statewide vs. regional permits
- Policy consistency requirements
- Funding flow from State to Regional
- Staff relationships
- Joint enforcement actions
