# PLAN-12-01: Adapt WaterBot.jsx to Dashboard Layout

## Objective
Adapt the standalone WaterBot.jsx to fit the vanderdev-website dashboard layout pattern, matching KiddoBot and BizBot conventions.

## Current State
- WaterBot.jsx exists in `/CA-AIDev/waterbot/src/pages/WaterBot.jsx`
- Uses standalone layout with its own `min-h-screen`, header, and footer
- Uses inline ReactMarkdown component
- Has its own session ID generation
- Chat webhook working: `https://n8n.vanderdev.net/webhook/waterbot`

## Target State
- WaterBot.jsx adapted to vanderdev-website patterns:
  - No `min-h-screen` wrapper (dashboard layout handles this)
  - Uses `BotHeader` component
  - Uses `useBotPersistence` hook for state management
  - Uses shared Icons from `../components/Icons`
  - Maintains all existing functionality

## Changes Required

### 1. Remove Standalone Layout Elements
- Remove outer `min-h-screen bg-slate-900` wrapper
- Remove custom header with logo/title

### 2. Add Shared Components
