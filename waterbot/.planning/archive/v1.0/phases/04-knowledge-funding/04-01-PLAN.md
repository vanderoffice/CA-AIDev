# Phase 4.1: State Revolving Funds

## Objective

Research and document California's State Revolving Fund (SRF) programs for water infrastructure financing.

## Research Topics

### Clean Water State Revolving Fund (CWSRF)
- Program overview and administration (SWRCB)
- Eligible project types (wastewater, stormwater, NPS)
- Funding terms (interest rates, repayment periods)
- Application process and timeline
- Priority ranking system (CWSRF Priority List)

### Drinking Water State Revolving Fund (DWSRF)
- Program overview and administration
- Eligible project types (treatment, distribution, storage)
- Small/disadvantaged community set-asides
- Principal forgiveness eligibility
- Funding agreements and disbursement

### Small Community Funding
- Technical assistance programs
- Planning and design funding
- Disadvantaged community criteria (MHI thresholds)
- SAFER program integration
