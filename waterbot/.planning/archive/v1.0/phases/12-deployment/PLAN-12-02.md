# PLAN-12-02: Integrate WaterBot into vanderdev-website

## Objective
Copy adapted WaterBot.jsx into vanderdev-website repository and wire up routing and navigation.

## Files to Modify in vanderdev-website

### 1. Copy New Files
- `src/pages/WaterBot.jsx` - adapted chat component

### 2. Update App.jsx
Add import and route:
```jsx
import WaterBot from './pages/WaterBot'

// In Routes:
<Route path="/waterbot" element={<WaterBot />} />
```

### 3. Update Sidebar.jsx
Add navigation item:
```jsx
// Import
import { ..., Droplets } from './Icons'

// In nav section (after KiddoBot):
<NavItem to="/waterbot" label="WaterBot" icon={Droplets} />
```

Also update MobileNav with same NavItem.
