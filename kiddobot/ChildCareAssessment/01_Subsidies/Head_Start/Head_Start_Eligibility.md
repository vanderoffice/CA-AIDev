# Head Start Eligibility Guide

**Purpose:** Determine if your family qualifies for Head Start or Early Head Start in California.

---

## Categorical Eligibility (Automatic)

These categories qualify regardless of income:

| Category | Documentation Needed | Notes |
|----------|---------------------|-------|
| **CalWORKs/TANF recipient** | Benefit verification letter | Any current receipt qualifies |
| **SSI recipient** | SSI award letter | Child OR parent receiving SSI |
| **Foster child** | Placement letter, court order | Includes kinship foster |
| **Homeless** | Self-declaration accepted | McKinney-Vento definition |
| **CalFresh/SNAP** | EBT card, benefit letter | Some programs accept |

### McKinney-Vento Homeless Definition

You're considered homeless if lacking a fixed, regular, adequate nighttime residence:
- Living in shelter, motel, or car
- Doubled up with others due to economic hardship
- Living in campground, park, or public space
- Awaiting foster care placement

**No documentation required** — self-declaration is accepted.

## Income Eligibility

### Federal Poverty Level (FPL) Guidelines

2026 Federal Poverty Guidelines (48 contiguous states):

| Family Size | 100% FPL (Annual) | 100% FPL (Monthly) |
|-------------|-------------------|-------------------|
| 1 | $15,960 | $1,330 |
| 2 | $21,640 | $1,803 |
| 3 | $27,320 | $2,277 |
| 4 | $33,000 | $2,750 |
| 5 | $38,680 | $3,223 |
| 6 | $44,360 | $3,697 |
| 7 | $50,040 | $4,170 |
| 8 | $55,720 | $4,643 |
| Each add'l | +$5,680 | +$473 |

*Source: HHS 2026 Poverty Guidelines (Federal Register, January 13, 2026). Updated annually each January.*

### How Income Is Calculated

| Included | Not Included |
|----------|--------------|
| Wages and salaries | CalFresh/SNAP benefits |
| Self-employment income | WIC benefits |
| Unemployment benefits | Housing assistance value |
| Social Security | One-time lump sums |
| Child support received | Student loans |
| Alimony | Earned Income Tax Credit |

### What Time Period?

Programs typically look at:
- **Previous 12 months** of income, OR
- **Previous calendar year** (tax return), OR
- **Current income** if significant change occurred

## Age Eligibility

### Head Start (Ages 3-5)

| Rule | Explanation |
|------|-------------|
| **Minimum age** | Must turn 3 by program's cutoff date |
| **Maximum age** | Until eligible for kindergarten |
| **Common cutoff** | September 1 (matches school year) |
| **Varies by program** | Some use December 1 cutoff |

### Early Head Start (Birth-3)

| Category | Age Range |
|----------|-----------|
| **Pregnant women** | From confirmation of pregnancy |
| **Infants** | Birth to 12 months |
| **Toddlers** | 12 months to 36 months |
| **Transition** | At 3, move to Head Start |

---

## Priority Enrollment

When programs have limited slots, priority goes to:

### Mandatory Reservations

| Category | Requirement |
|----------|-------------|
| **Children with disabilities** | 10% of enrollment |
| **Homeless children** | Priority enrollment |
| **Foster children** | Priority enrollment |

### Selection Criteria

Programs use point-based systems considering:

| Factor | Higher Priority |
|--------|-----------------|
| **Income** | Lowest income first |
| **Parent work/school** | Working or in training |
| **Single parent** | Additional points |
| **Teen parent** | Additional points |
| **Disability** | Child or family member |
| **Previous enrollment** | Returning families |

---

## Over-Income Enrollment

Programs **may** enroll families above FPL when:

### 130% FPL Exception

| Family Size | 130% FPL (Annual) |
|-------------|-------------------|
| 2 | $27,495 |
| 3 | $34,645 |
| 4 | $41,795 |
| 5 | $48,945 |

Up to 35% of enrollment can be families between 100-130% FPL.

### Over-Income Exceptions

| Circumstance | Explanation |
|--------------|-------------|
| **After all eligible served** | Empty slots can go to over-income |
| **Child has disability** | 10% slot can go to any income |
| **Community need** | Documented local circumstances |
| **Sibling enrolled** | Some programs prioritize siblings |

## Residency Requirements

### Where You Live Matters

| Requirement | Details |
|-------------|---------|
| **Service area** | Must live in program's geographic area |
| **Documentation** | Utility bill, lease, or ID |
| **Homeless exception** | Location of shelter/services accepted |
| **Moving** | May need to transfer programs |

### If You Move

- Notify current program immediately
- Request transfer to new area's program
- Records will be forwarded
- May face waitlist at new location

## Special Circumstances

### Child with Disability

| Situation | What Happens |
|-----------|--------------|
| **Suspected delay** | Program will screen and refer for assessment |
| **Already has IEP/IFSP** | Provide documentation; priority enrollment |
| **Over-income** | May still qualify for 10% disability slots |
| **Needs accommodations** | Program must provide or arrange |

### Teen Parents

| If You Are... | You May Qualify For... |
|---------------|------------------------|
| Under 18 | Head Start for your child |
| In school | Programs accommodating school schedules |
| Both | Your child and yourself (family literacy) |

### Grandparents/Kinship Caregivers

| Situation | Eligibility |
|-----------|-------------|
| Legal guardian | Based on your household income |
| Informal care | May use parent's income |
| Foster/kinship | Automatic categorical eligibility |

## Application Process

### Step 1: Contact Local Program

Find your local Head Start:
- [Head Start Locator](https://eclkc.ohs.acf.hhs.gov/center-locator)
- [Head Start California](https://headstartca.org)
- Call your local CCR&R

### Step 2: Complete Application

| What to Expect |
|----------------|
| Application form (online or paper) |
| Interview with family services staff |
| Documentation submission |
| Home visit (in some programs) |

### Step 3: Eligibility Determination

| Outcome | Next Steps |
|---------|------------|
| **Eligible + slot available** | Enrollment begins |
| **Eligible + no slot** | Placed on waitlist |
| **Not eligible** | Referral to other programs |

### Timeline

| Step | Typical Timeframe |
|------|-------------------|
| Application review | 1-2 weeks |
| Documentation gathering | Varies |
| Enrollment notification | When slot available |
| Program start | Beginning of term or rolling |

## If You're Not Eligible

### Alternative Programs

| Program | Who Qualifies |
|---------|---------------|
| **California State Preschool (CSPP)** | Higher income limit (100% SMI) |
| **Subsidized childcare (APP)** | Working families up to 85% SMI |
| **Transitional Kindergarten** | 4-year-olds (free, no income limit) |
| **Migrant Head Start** | Farmworker families |

### Appeals

If you believe you were wrongly denied:
1. Request written explanation
2. Ask about appeal process
3. Contact program director
4. Contact Head Start regional office if unresolved
