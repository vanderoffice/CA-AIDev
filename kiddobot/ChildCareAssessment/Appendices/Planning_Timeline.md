## First Year of Life (Birth to 12 Months)

### Month 1-3: Parental Leave

| If You Need... | Do This | When |
|----------------|---------|------|
| Care soon | Contact waitlists to confirm status | Week 4 |
| Delayed return | Update providers on new start date | When known |
| To apply now | Begin applications during leave | ASAP |

### Month 4-6: Preparing for Care Start

| If Starting Care Soon | Action |
|----------------------|--------|
| 2 weeks before | Confirm start date with provider |
| 1 week before | Complete final enrollment paperwork |
| 1 week before | Schedule pediatrician for required forms |
| 3 days before | Do a practice run (visit, short stay) |
| Day 1 | Begin with shorter day if possible |

### Month 6-12: During Infant Care

| Ongoing Action | Frequency |
|----------------|-----------|
| Communicate with provider | Daily pickup/dropoff |
| Update on milestones | Monthly |
| Review authorization (if subsidized) | Every 6 months |
| Check immunization schedule | Per pediatrician |
| Start planning toddler transition | Month 9-10 |

## Year 1-2: Infant to Toddler

### Key Transition Points

| Age | Common Transition | What to Know |
|-----|-------------------|--------------|
| 12 months | Infant room → Toddler room | Different ratios (1:4 or 1:6) |
| 18 months | Walking milestone | Some programs require this for transition |
| 24 months | Toddler programs begin | CSPP age-eligible children |

### Planning the Toddler Transition

**Start at 9-12 months:**
1. [ ] Ask current provider about their transition age
2. [ ] Tour toddler room if same provider
3. [ ] Research new providers if changing care
4. [ ] Apply for toddler programs with waitlists
5. [ ] Update subsidy authorization if needed

## Year 2-3: Preparing for Preschool

### At Age 2 (24 Months)

| Action | Why |
|--------|-----|
| Research preschool options | Enrollment periods vary |
| Understand eligibility changes | Some programs start at 2, others at 3 |
| Apply for CSPP (if eligible) | Spring enrollment for fall start |
| Apply for Head Start (if eligible) | Priority given to 4-year-olds, so apply early |

### Between 2 and 3

| Timeline | Action |
|----------|--------|
| 6 months before age 3 | Tour preschool programs |
| 4-5 months before | Apply for CSPP, Head Start |
| 3 months before | Complete enrollment process |
| 1 month before | Prepare child for transition |

## Year 3-4: Preschool Years

### At Age 3

**Major programs become available:**

| Program | Eligibility | Application |
|---------|-------------|-------------|
| Head Start | 100% FPL or categorical | Local grantee |
| CSPP | 100% SMI (~$90K for family of 3) | Local contractor |
| TK (some districts) | Turning 4 by specific date | School district |

### Preschool Application Timeline

```
JANUARY        FEBRUARY       MARCH          APRIL          AUGUST
    │              │              │              │              │
    ├──────────────┼──────────────┼──────────────┼──────────────┤
    │              │              │              │              │
RESEARCH      TOUR           APPLY          ACCEPT          START
    │              │              │              │              │
    ↓              ↓              ↓              ↓              ↓
Identify      Visit          Submit         Confirm        Begin
programs      facilities     applications   enrollment     preschool
```

### At Age 3-4: Special Needs Transition

**If child has IFSP (Early Start):**

| Timeline | Action |
|----------|--------|
| 2 years, 9 months | Regional Center refers to school district |
| 2 years, 10 months | School district evaluates for IEP |
| Before 3rd birthday | Transition meeting |
| 3rd birthday | Services transfer to school district |

## Year 4-5: Transitional Kindergarten

### Universal TK Eligibility (2025-26)

**Your child is eligible for TK if they turn 4:**
- Between September 2 and February 2 (2024-25 school year)
- Expanding annually until all 4-year-olds eligible by 2025-26

### TK Timeline

| When | Action |
|------|--------|
| Fall before TK | Research TK at local schools |
| January-February | Attend kindergarten roundup/registration |
| February-March | Submit TK enrollment |
| Spring | Complete enrollment paperwork |
| August-September | TK begins |

### TK + Childcare Combination

TK is typically only 3-4 hours. For working parents:

| Need | Options |
|------|---------|
| Before TK | Before-school care at school or provider |
| After TK | Extended day program, ASES, private care |
| Full day | Some districts offer "expanded TK" (full day) |
| Summer | Summer camp, childcare provider |

## Subsidy Program Deadlines

### Annual Recertification

All subsidized care requires recertification every 12 months.

| Program | Recertification Window | What Happens If Missed |
|---------|------------------------|------------------------|
| CalWORKs Stages | Tied to cash aid review | Care may terminate |
| APP | 12 months from enrollment | 30-day grace period, then termination |
| CSPP | Typically before fall term | May lose slot |
| Head Start | Annual enrollment | May need to reapply |

**Mark your calendar:** Recertification due date: ____________

### Income/Change Reporting

| Change Type | Report Within | How to Report |
|-------------|---------------|---------------|
| Income increase/decrease | 5 days | Call APP contractor |
| Job change | 5 days | Call APP contractor |
| Address change | 5 days | Call APP contractor |
| Provider change | Before switching | Get new authorization |
| Family size change | 5 days | Call APP contractor |

## Key Application Windows

### Head Start

| Month | Activity |
|-------|----------|
| October-November | Applications open for following year |
| February-April | Most programs notify acceptances |
| August-September | Program year begins |
| Year-round | Rolling enrollment if slots available |

### CSPP

| Month | Activity |
|-------|----------|
| January-March | Applications accepted |
| April-May | Eligibility determinations |
| August-September | School year begins |
| Year-round | Some programs have rolling enrollment |

### Transitional Kindergarten / Kindergarten

| Month | Activity |
|-------|----------|
| January-February | Kindergarten roundup (info sessions) |
| February-March | Enrollment opens |
| March-April | Priority enrollment deadline |
| May | Late enrollment still possible |
| August | School year begins |

## Emergency Planning

### What to Do Before a Crisis

1. [ ] **Identify 3 backup care options** (family, friends, backup provider)
2. [ ] **Keep provider list updated** (who might have emergency openings)
3. [ ] **Know your CCR&R contact** (for emergency referrals)
4. [ ] **Save 211 number** (crisis resources)
5. [ ] **Document your eligibility** (keep copies of key paperwork)

## Checklists by Stage

### Expecting or Newborn
- [ ] Contact CCR&R for consultation
- [ ] Research care types and costs
- [ ] Check subsidy eligibility
- [ ] Apply for subsidies if eligible
- [ ] Tour providers
- [ ] Join waitlists
- [ ] Confirm enrollment
- [ ] Complete all paperwork
- [ ] Know backup care plan

### Toddler (1-3)
- [ ] Monitor for toddler room transition
- [ ] Research preschool options
- [ ] Apply for preschool programs
- [ ] Maintain subsidy recertification
- [ ] Request Regional Center assessment (if concerns)
- [ ] Plan for potty training transition (some programs require)

### Preschool Age (3-5)
- [ ] Enroll in preschool program
- [ ] Apply for TK if age-eligible
- [ ] Plan before/after school care
- [ ] Prepare for kindergarten registration
- [ ] Know IEP timeline if applicable

### School Age (5+)
- [ ] Arrange before/after school care
- [ ] Research summer programs
- [ ] Know school break schedule
- [ ] Understand subsidy changes (if any)
- [ ] Plan for program transitions
