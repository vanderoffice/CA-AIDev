# CA-AIDev

Multi-agent AI systems and citizen-facing government services.

## Structure

```
CA-AIDev/
├── shared/              # Shared resources across projects
├── bizbot/              # Business licensing multi-agent system
├── commentbot/          # Public comment analysis system
├── adabot/              # ADA compliance guidance
├── askca/               # Digital services research
│   └── domain-crawler/  # Government endpoint crawler
└── kiddobot/            # Childcare services (in development)
```

## Projects

### BizBot
Multi-agent n8n system providing personalized California business licensing guidance. Coverage includes all 482 CA cities and 58 counties.

### CommentBot
Multi-agent system for analyzing public comments submitted to state government agencies using specialized legal and scientific agents.

### ADABot
Guide and assessment system for ADA (Americans with Disabilities Act) compliance.

### AskCA
Government digital services research and documentation. Includes the Domain Crawler for discovering California state government endpoints.

### KiddoBot
Agentic architecture to make finding childcare easier (in development).
